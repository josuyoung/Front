테이블 <table>
여러 종류의 데이터를 보기 좋게 정리하여 보여주는 표

<table>
    <tr> <!-- 테이블의 각층 -->
        <td></td> <td></td> <!-- 테이블의 각 셀-->
    </tr>
</table>

<table>속성
- border :  테이블의 테두리 두께를 설정(기본 0)
- width : 테이블의 가로폭을 결정
- align : 테이블의 정렬을 설정(left,right,center)
<hr> 태그
- 가로선 그어줌

- width : 셀의 가로 폭을 설정
- height : 셀의 높이를 설정
- align : 셀의 텍스트 정렬 설정(left,right,center)

colspan 속성
- 가로로 열을 합칠 때 사용

    <td colspan = "합칠 열의 개수">

rowspan 속성
- 세로로 행을 합침

    <td rowspan = "합칠 행의 개수">

<colgroup>
- <colgroup> 태그 뒤에 나오는 컬럼(th 또는 td)에 적용할 스타일을 해당 태그에서 미리 설정할 수 있도록 함

<caption>
- 표에 제목을 붙일 때 사용
- 기본 위치는 표의 상단 중앙

    <a href="a 태그의 #값 또는 id속성의 값">링크에 사용될 문자 또는 이미지</a>
